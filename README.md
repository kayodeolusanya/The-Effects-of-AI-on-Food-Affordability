## üìå Summary of the Data Story

When COVID-19 disrupted global supply chains, food systems became unstable worldwide. As shelves emptied and prices spiked, industries and governments increasingly turned to **AI-powered forecasting, logistics optimization, and automation** to restore stability.

This project uses the pandemic as a **benchmark** to evaluate what changed and how AI influenced the recovery period.

---

## üß† Key Insights

* Some food categories became more stable due to AI interventions - but not all.
* **Post-AI (2022-2025)** volatility increased for several essential staples.
* Efficiency gains from automation did **not always translate into lower prices**.
* Technological access remains uneven - **small producers benefitted less**.
* Price trends reveal a **consistent upward shift** across many food categories after 2021.
